### 介绍

后台管理模板框架，[领浩科技](https://linghao100.com)出品。

在Vue、React时代，如果你还想使用JQuery、Bootstrap相关技术栈，那么LcukAdminTpl模板框架可能是你一个不错的选择。

LcukAdminTpl 提供常规版和多菜单版本。

基于 Bootstrap5 开发。

样式布局参考蚂蚁金服 Ant Design Pro 框架。
